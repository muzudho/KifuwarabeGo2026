# 続きはここから

最終更新: 2026-07-20

## 現在の状態

- ソリューションは `KifuwarabeGo2026.Gui`、`KifuwarabeGo2026.Engine`、`KifuwarabeGo2026.Shared`、`KifuwarabeGo2026.Gui.Communication.Cgos` の4プロジェクト構成。
- `GoBoard`、`GoPoint`、`GoStone`、`GoRenParseResult` はSharedへ移動済み。
- GUIとEngineはSharedを参照し、EngineからGUIへの依存はない。
- CGOS通信クライアントはGUIが外部プロセスとして起動する。
- 利用者向け配布物はGUI版とEngine版の2種類だけ。
- GUIをpublishすると、CGOS通信コンポーネントが `Tools\Cgos` へ自動的にpublishされる。
- SharedはGUI版とEngine版へ自動的に含まれるため、単独publish・単独配布は不要。

## リリース手順

最初に [開発/リリース手順.md](./開発/リリース手順.md) を読む。

リリース用コマンドは次の2本。

```powershell
dotnet publish KifuwarabeGo2026.Gui\KifuwarabeGo2026.Gui.csproj -c Release -r win-x64 --self-contained false
dotnet publish KifuwarabeGo2026.Engine\KifuwarabeGo2026.Engine.csproj -c Release -r win-x64 --self-contained false
```

CGOSとSharedを個別にpublishしないこと。
