# Kifuwarabe Go 2026 v1.1.0

CGOSへの接続・対局観戦から棋譜レビューまでの導線を追加し、ローカル対局画面と各種設定画面の使い勝手を改善しました。

## 主な変更

- CGOS接続設定と管理接続、プレイヤー接続に対応
- CGOS対局の観戦と、終局後の棋譜レビューに対応
- 黒番・白番で別々のGTPエンジンを選択可能に変更
- GTPエンジンのGUIオプション設定画面を追加
- SGFの保存、読み込み、局面編集、棋譜レビューを改善
- 対局者名、手番、思考状態などの表示を改善
- 星付近を優先するランダム合法手エンジンを追加
- CGOSクライアント画面とローカル対局設定画面のレイアウトを改善
- 各種ボタン表記と操作導線を整理

## 動作環境

- Windows
- .NET 8 Desktop Runtime
