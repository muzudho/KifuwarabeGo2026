# Kifuwarabe Go 2026 v2.0.0

コンピューター囲碁フォーラム主催のCGFオープンへ持参する大会用メジャーリリースです。CGOSへ黒番・白番のエンジンを接続する設定と、国際式の盤座標表示を中心に仕上げました。

## 主な変更

- CGOS接続画面で黒番・白番のGTPエンジンを個別に選択可能
- 黒番・白番ごとにCGOSログイン名とパスワードを接続前に編集可能
- エンジン表示名とは別に、CGOSのデフォルトログイン名と平文パスワードを保存可能
- エンジン選択、ログイン名、パスワードの順に操作しやすい接続画面へ整理
- CGOS通信ログではパスワード本文を伏せて記録
- CGOS画面から開いたGTPエンジン設定でも、すべてのテキスト欄を編集可能
- 盤座標を左下原点の国際式へ変更
- 列文字を盤の下辺へ移し、`I` を飛ばす表記に対応
- 行番号を盤の左辺へ下から上に増える順で表示
- 盤座標の配色と石の影との間隔を調整

## 配布物

- GUI版: `KifuwarabeGo2026.Gui-v2.0.0-win-x64.zip`
- Engine版: `KifuwarabeGo2026.Engine-v2.0.0-win-x64.zip`

GUIとEngineの両方を使う場合は、2つともダウンロードしてください。

## 動作環境

- Windows x64
- .NET 8 Desktop Runtime
