# 続きはここから

最終更新: 2026-07-12

## 現在の入口

- 次回は GUI の `Connect To CGOS` メニューから先を作る。
- いまは起動直後に `Local (推奨)` / `Connect To CGOS` を選べる。
- `Connect To CGOS` を押すと CGOS 接続先リスト画面へ進む。
- 接続先リストは `練習`、`2026年大会予選`、`2026年大会本戦` のようなプロファイルを持てる。
- CGOS へ実際に接続して観戦・対局へ進む GUI 側の画面遷移は、まだ未実装。

## 今日できたこと

- CGOS 用の独立プロジェクト `KifuwarabeGo2026.Communication.Cgos` を追加済み。
- `KifuwarabeGo2026.Communication.Cgos` は `uec-go.com:6809` へ接続し、黒番 `KifuwarabeB` / 白番 `KifuwarabeW` でログインできる。
- 端末で `Ctrl+C` を押すと CGOS へ `quit` を送ってログアウトする。
- GUI の最初の選択画面に `Local (推奨)` と `Connect To CGOS` を追加した。
- 文字が読めない場合でも囲碁系の選択肢だと分かるよう、ローカル用と CGOS 用の箱アイコンを調整した。
- CGOS 接続先リスト画面に `PREV`、`NEXT`、`ADD`、`EDIT`、`DUPLICATE`、`DELETE` を追加した。
- CGOS 接続先プロファイルの追加、編集、複製、削除、ページ移動ができる。
- 編集パネルで `DISPLAY NAME`、`HOST`、`PORT`、`ROLE`、`NOTE` を編集できる。
- CGOS 接続先プロファイルは JSON として `KifuwarabeGo2026/Content/CgosConnections/cgos-connection-list.json` に保存される。

## 主な実装場所

- GUI 入口とクリック処理: `KifuwarabeGo2026/Game1.cs`
- 画面描画とヒット判定: `KifuwarabeGo2026/Presentation/GoScreenRenderer.cs`
- CGOS 接続先の状態管理: `KifuwarabeGo2026/Application/GoAppSession.cs`
- CGOS 接続先データ: `KifuwarabeGo2026/Application/CgosConnectionProfile.cs`
- CGOS 接続先 JSON 読み書き: `KifuwarabeGo2026/Application/CgosConnectionCatalog.cs`
- CGOS 接続先 JSON: `KifuwarabeGo2026/Content/CgosConnections/cgos-connection-list.json`
- CGOS 通信クライアント: `KifuwarabeGo2026.Communication.Cgos/Program.cs`
- CGOS 通信クライアント説明: `KifuwarabeGo2026.Communication.Cgos/README.md`
- 個人用メモ: `KifuwarabeGo2026/Docs/【むずでょ個人用】/【むずでょ個人用】CGOSとの通信について.md`
